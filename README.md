# Rewire — Behavioral Change App

> Break bad habits. Build better ones. One drill at a time.

**Status:** Live

## What it does
- Micro-drills for improving focus and attention
- Habit tracking with streak system
- Support for breaking addictions through structured routines
- Clean, distraction-free UI with smooth animations

## Tech Stack
`React Native` `TypeScript` `Supabase` `Framer Motion` `Expo`
